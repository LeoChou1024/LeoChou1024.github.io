前言

箭頭函數(Fat arrow functions）是一個來自ECMAScript 2015（又稱ES6）的全新特性。

簡單的來說，是ES6版才有的新功能

箭頭函數的產生，主要為：更簡潔的語法 讓父類別共享關鍵字this。

這樣講可能有些許模糊，我們帶個下面的例子讓大家看一下就知道了。
